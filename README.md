# HEALTHCARE-SALES-DASHBOARD
The Healthcare Sales Intelligence Dashboard is an interactive Power BI report designed to analyze and monitor key business metrics in the healthcare/pharmaceutical domain. It provides actionable insights into sales performance, patient trends, product performance, and departmental profitability.

This dashboard is built with a modern dark-themed UI inspired by SaaS products, focusing on clarity, usability, and data storytelling.

🎯 Objectives
Track overall sales and profit performance
Analyze patient distribution (Sales vs Returns)
Identify top-performing drugs
Compare department-wise performance
Enable dynamic filtering for deeper insights
📊 Key Metrics (KPIs)
💰 Total Sales: 236M
📈 Total Profit: 45M
👤 Total Patients: 14K
📦 Total Net Quantity: 711K
📈 Dashboard Features
1. 📊 Sales vs Profit Analysis
Combined bar + line chart
Tracks sales and profit across drug categories
Helps identify high-revenue but low-profit products
2. 🍩 Patient Distribution
Donut chart showing Sales vs Return patients
Provides quick insight into return rate and customer behavior
3. 💊 Top Drugs Performance
Highlights top-performing drugs by sales
Enables comparison of product contribution
4. 🏢 Department Analysis
Quantity vs Return Quantity comparison
Profit distribution by department
Identifies high-performing and underperforming departments
5. 🎛️ Interactive Filters
Drug Name
Sub Category
Sub Category Level

👉 Allows users to dynamically explore and drill down into data
